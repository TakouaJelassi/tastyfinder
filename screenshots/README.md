# Screenshots

Place app screenshots here with these exact filenames so they show up in the main README:

| Filename | Page |
|---|---|
| `home.png` | Home / recipe search |
| `detail.png` | Recipe detail |
| `chat.png` | AI chat assistant |
| `generate.png` | AI recipe generator |
| `shopping.png` | Shopping list |
| `profile.png` | User profile |

**Tips for clean screenshots:**
- Use a browser window around 1280×800 for desktop shots
- Hide personal data (real email) where possible
- PNG format, ideally compressed

Once added:

```bash
git add screenshots/
git commit -m "docs: add app screenshots"
git push
```
